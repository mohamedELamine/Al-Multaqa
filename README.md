# Al-Multaqa WordPress Theme

قالب WordPress متعدد الاستخدام متخصص للشركات العربية مع دعم RTL كامل.

## الميزات

- **RTL-First**: دعم RTL أصلي، ليس إضافة
- **سريع**: تحميل فائق (90+ PageSpeed)
- **WooCommerce جاهز**: متوافق مع المتاجر
- **Elementor**: Page Builder جاهز
- **خطوط عربية**: Tajawal و Cairo
- **Responsive**: يعمل على كل الأجهزة
- **Accessibility**: WCAG AA متوافق

## التثبيت

1. ارفع مجلد `Al-Multaqa` إلى `/wp-content/themes/`
2. فعّل القالب من WordPress Admin
3. اتبع معالج الإعداد

## التخصيص

- الألوان: من خلال Customizer
- القوائم: Appearance > Menus
- السايدبار: Appearance > Widgets
- الشعار: Appearance > Customize

## الاعتمادات

- WordPress 5.8+
- PHP 7.4+
- WooCommerce (اختياري)
